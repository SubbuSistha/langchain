# ReAct in LangChain

## 1. What is ReAct?

**ReAct** stands for **Reasoning + Acting**.

It is a prompting and agent design technique where a language model: 1.
**Reasons about a problem step‑by‑step** 2. **Takes actions using
tools** 3. **Observes the results** 4. **Continues reasoning until it
reaches the final answer**

This approach helps LLMs solve complex tasks that require **external
information, APIs, or tools**.

The idea was introduced in the paper:

> *ReAct: Synergizing Reasoning and Acting in Language Models*

LangChain uses this concept to build **agents that think and interact
with tools**.

------------------------------------------------------------------------

# 2. Why ReAct is Important

Normal LLM prompting:

User Question → LLM → Final Answer

Problem: - The model cannot access **external data** - Cannot perform
**search, calculations, API calls** - Cannot verify its reasoning

ReAct solves this by allowing the model to:

-   Think (Reasoning)
-   Use Tools (Actions)
-   Observe results
-   Continue thinking

Flow:

User Question → Reason → Tool → Observation → Reason → Final Answer

------------------------------------------------------------------------

# 3. ReAct Workflow

A ReAct agent follows a loop:

Thought → Action → Observation

Example:

Question: "Who is the CEO of the company that created LangChain?"

Reasoning chain:

Thought: I need to find the company behind LangChain.

Action: Search("LangChain company")

Observation: LangChain was created by LangChain Inc.

Thought: Now I need to find the CEO.

Action: Search("CEO of LangChain Inc")

Observation: Harrison Chase

Final Answer: Harrison Chase

------------------------------------------------------------------------

# 4. ReAct Prompt Structure

ReAct prompts typically follow this format:

    Question: <user question>

    Thought: reasoning step
    Action: tool to call
    Action Input: tool input

    Observation: result from tool

    Thought: continue reasoning
    ...
    Final Answer: answer

This structured pattern allows the model to: - break down problems - use
tools step-by-step

------------------------------------------------------------------------

# 5. ReAct in LangChain

LangChain provides built-in support for ReAct agents.

Example tools: - Web Search - Calculator - Database Query - APIs

The agent automatically decides:

1.  When to think
2.  Which tool to call
3.  How to use the result

------------------------------------------------------------------------

# 6. Example Code (LangChain ReAct Agent)

``` python
from langchain.agents import initialize_agent
from langchain.agents import AgentType
from langchain_openai import ChatOpenAI
from langchain.tools import Tool

llm = ChatOpenAI()

tools = [
    Tool(
        name="Calculator",
        func=lambda x: eval(x),
        description="Useful for math calculations"
    )
]

agent = initialize_agent(
    tools,
    llm,
    agent=AgentType.REACT_DESCRIPTION,
    verbose=True
)

agent.run("What is (25 * 4) + 10?")
```

Output reasoning may look like:

Thought: I should calculate 25 \* 4 Action: Calculator Action Input:
25\*4

Observation: 100

Thought: Now add 10 Action: Calculator Action Input: 100+10

Observation: 110

Final Answer: 110

# 7. Advantages of ReAct

1.  **Better reasoning**
2.  **Tool usage**
3.  **Step-by-step transparency**
4.  **Reduced hallucination**
5.  **More controllable agents**

------------------------------------------------------------------------

# 8. ReAct vs Chain-of-Thought

| Feature \| Chain-of-Thought \| ReAct \|

\|-------\|-------\|------\|Reasoning\|Yes\|Yes\| \|Tool
Usage\|No\|Yes\| \|External Data\|No\|Yes\| \|Agent Behavior\|No\|Yes\|

Chain-of-Thought = reasoning only

ReAct = reasoning + tools

------------------------------------------------------------------------

# 9. Real Use Cases

ReAct agents are used for:

-   AI Assistants
-   Autonomous agents
-   Customer support bots
-   Research assistants
-   Data retrieval systems
-   Code assistants

Example:

User: "Find today's weather in Hyderabad and convert it to Fahrenheit."

Agent flow:

Thought → Search weather\
Action → Weather API\
Observation → 30°C

Thought → Convert to Fahrenheit\
Action → Calculator\
Observation → 86°F

Final Answer → 86°F

------------------------------------------------------------------------

# 10. Key Takeaway

ReAct is a **core concept behind modern AI agents**.

It enables LLMs to:

-   Think step-by-step
-   Use tools
-   Interact with external systems
-   Solve complex tasks

In LangChain, ReAct is commonly used for **agent frameworks and
tool-based AI systems**.
